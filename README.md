# CSE Stock Dashboard

This project is a personal stock market analysis dashboard for the Sri Lankan market.

## Structure
- `/server`: Node.js/TypeScript backend with SQLite.
- `/client`: React/Vite frontend with Lightweight Charts.

## Quick Start
1. **Server**: `cd server && npm install && npm run seed && npm run dev`
2. **Client**: `cd client && npm install && npm run dev`

See [walkthrough.md](./walkthrough.md) for more details.
